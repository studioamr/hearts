# HEARTS — Battle Royale

Un battle royale de arqueros estilo TowerFall donde cada corazón que cargas se puede robar.
8 entran. Uno se lo lleva todo.

**Juega gratis en el navegador** · sin descargas.

- 28 guerreros (depredadores vs presas), cada uno con su rareza y poder
- Cofres de tesoro (gacha) para conseguir guerreros — el primero es gratis
- 4 mundos: selva, desierto, norte helado y volcán
- Arcos con flechas limitadas que recuperas del suelo (homenaje a TowerFall)

Hecho en Morelia, México. Sonido y música: Juhani Junkala (CC0, OpenGameArt).
